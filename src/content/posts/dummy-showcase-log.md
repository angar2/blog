## 헤딩과 채번

이 헤딩에는 md 원문에 번호가 없다. 렌더링된 화면에서 `1.` 이 자동으로 붙고 하단에 가로 구분선이 보이면 채번 장치가 작동하는 것이다.

### 하위 헤딩

#### 더 하위 헤딩

> [!note]
> note 콜아웃이다. GitHub에서 열어도 의미가 보존된다.

> [!warn]

## 강조 4계층과 코드

**핵심명사**는 볼드, `Identifier`는 코드, <u>조건이 성립하는 경우에만</u> 밑줄, *참고 서지*는 이탤릭로 쓴다.

```ts title="src/lib/example.ts" showLineNumbers
export function greet(name: string): string {
  // 파일명 라벨과 줄번호 지정이 함께 걸린 코드 블록
  return `hello, ${name}`;
}
```
